# ハコ割り生成ツール クイックセットアップ

## 必要なもの（事前準備）

| 必要なもの | 入手先 | 備考 |
|-----------|--------|------|
| **Docker Desktop** | https://www.docker.com/products/docker-desktop/ | 無料。Windows / Mac 両対応 |
| **ハコ割り生成ツール** | 配布ファイル | EXE または DMG |

---

## Windows へのインストール

1. **Docker Desktop** をインストールし、起動しておく
2. **ハコ割り生成ツール_setup.exe** をダブルクリック
3. セットアップウィザードの「次へ」を進めてインストール完了
4. デスクトップの **「ハコ割り生成ツール 起動」** をダブルクリック

---

## Mac へのインストール

1. **Docker Desktop** をインストールし、起動しておく
2. **ハコ割り生成ツール_setup.dmg** を開く
3. **ハコ割り生成ツール.app** を **Applications** フォルダへドラッグ
4. Finder の Applications から **ハコ割り生成ツール.app** を **右クリック → 開く**
   - ※ 初回のみ Gatekeeper の警告が出ます。「開く」を選択してください。

---

## 初回起動について

> **初回のみ** Whisper AI モデル（約 800 MB）のダウンロードが発生します。
> インターネット接続により **5〜15 分** かかります。
> 2 回目以降は **数秒** で起動します。

進捗はコマンドウィンドウ（Windows）またはターミナル（Mac）に表示されます。
ブラウザが自動で開いたら準備完了です。

---

## 基本的な使い方（3 ステップ）

```
STEP 1  音声・映像ファイルをアップロード
         対応形式: MP3, WAV, MP4, MPG, TS, M4A, AAC, FLAC など

STEP 2  ハコ割りテキスト (.txt) をアップロード
         空行区切り または 改行区切り を選択

STEP 3  「▶ SRT を生成」をクリック
         → SRT ファイルをダウンロード
```

**出力先**: デスクトップの `ハコ割り生成ツール_output` フォルダ

---

## アプリの停止

| OS | 方法 |
|----|------|
| Windows | デスクトップの「ハコ割り生成ツール 停止」をダブルクリック |
| Mac | `ハコ割り生成ツール_停止.app` をダブルクリック |

---

## こんなときは？

| 症状 | 対処 |
|------|------|
| ブラウザが開かない | http://localhost:8503 を手動で開く |
| Docker エラーが出る | Docker Desktop が起動しているか確認 |
| 処理が遅い | ローカル処理は CPU のみ使用。長い動画は時間がかかります |
| API キーを使いたい | サイドバーで「OpenAI API」を選択し、キーを入力 |

詳しくは **マニュアル.md** を参照してください。
