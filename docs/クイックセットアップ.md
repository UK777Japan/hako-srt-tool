# ハコ割り生成ツール　クイックセットアップ

---

## ▶ まず Docker Desktop をインストール（必須・無料）

| OS | ダウンロード先 |
|----|--------------|
| Windows | https://www.docker.com/products/docker-desktop/ → **Download for Windows** |
| Mac (Apple Silicon) | https://www.docker.com/products/docker-desktop/ → **Download for Mac (Apple Silicon)** |
| Mac (Intel) | https://www.docker.com/products/docker-desktop/ → **Download for Mac (Intel)** |

インストール後、Docker Desktop を起動し、タスクバー（Windows）またはメニューバー（Mac）に  
クジラのアイコン 🐳 が表示されるのを確認してください。

---

## ▶ ハコ割り生成ツールをインストール

### Windows の場合

```
1. ハコ割り生成ツール_setup.exe をダブルクリック
2. 「次へ」を数回クリックしてインストール完了
3. デスクトップの「ハコ割り生成ツール 起動」をダブルクリック
```

### Mac の場合

```
1. ハコ割り生成ツール_setup.dmg を開く
2. ハコ割り生成ツール.app を Applications フォルダへドラッグ
3. Finder の Applications から「ハコ割り生成ツール.app」を
   右クリック → 「開く」（初回のみ警告が出ます）
```

---

## ▶ 初回起動について

> **初回だけ** Whisper AI モデル（約 800 MB）をダウンロードします。  
> インターネット回線により **5〜15 分** かかります。  
> ブラウザが自動で開いたら準備完了です。

---

## ▶ 基本の使い方（3 ステップ）

**① 音声・映像ファイルをアップロード**

対応形式: `MP3` `WAV` `MP4` `MPG` `TS` `M4A` `AAC` `FLAC` `OGG`

**② ハコ割りテキスト（.txt）をアップロード**

```
【空行区切りの例】           【改行区切りの例】
本日はお越しいただき         本日はお越しいただきありがとうございます。
ありがとうございます。        本日のテーマをご説明します。
                             よろしくお願いいたします。
本日のテーマを
ご説明します。
```

**③「▶ SRT を生成」をクリック**

完了後に SRT・TXT・CSV の 3 種類をダウンロードできます。

---

## ▶ 出力ファイルの保存先

```
デスクトップ
└── ハコ割り生成ツール_output/
    ├── output.srt   ← 字幕ファイル（Premiere / DaVinci / YouTube に対応）
    ├── output.txt   ← テキスト確認用
    └── output.csv   ← Excel で開ける表形式
```

---

## ▶ 困ったときは

| 症状 | 対処 |
|------|------|
| ブラウザが開かない | http://localhost:8503 を直接入力 |
| Docker エラー | Docker Desktop が起動しているか確認 |
| 処理が遅い | 長い動画は時間がかかります（目安: 60分動画 → 約 15〜25 分） |
| 文字化け | テキストファイルを UTF-8 で保存し直す |

詳しくは **マニュアル.md** をご覧ください。
