QUICK SETUP GUIDE

📄 ハコ割り生成ツール

音声・映像ファイルとハコ割りテキストから SRT 字幕を自動生成するツール

1
インストール
🪟 Windows（Docker 不要）
  1. ハコ割り生成ツール_setup.exe をダブルクリック
  2. 「次へ」を数回クリックしてインストール完了
  3. デスクトップの 「ハコ割り生成ツール 起動」 をダブルクリック
  4. 自動でブラウザが開いたら準備完了

Python・ffmpeg・Whisper AIモデルをすべて同梱。追加インストール一切不要。

🍎 Mac（事前準備が必要）
  1. docker.com から Docker Desktop をインストール・起動する
  2. ハコ割り生成ツール_setup.dmg を開く
  3. アプリを Applications フォルダへドラッグ
  4. Finder の Applications から 右クリック → 「開く」
    （初回のみ Gatekeeper 警告が出ます）
2
初回起動について
Windows Whisper AI モデルはインストーラーに同梱済みです。
追加のダウンロードなく、インストール後すぐに使い始めることができます。
2回目以降 数秒で起動します。ブラウザが自動で開いたら準備完了です。
3
基本の使い方
01

音声・映像ファイルをアップロード

対応形式:

MP3WAVMP4 MPGTSM4A AACFLACOGG
02

ハコ割りテキスト (.txt) をアップロード

UTF-8 で保存されたテキストファイルを選択します。
区切り方式（空行 / 改行）を選んでください。

03

「▶ SRT を生成」をクリック

完了後に SRT・TXT・CSV の 3 種類をダウンロードできます。

A/B
ハコ割りテキストの形式
✅ 空行区切り（ブロック間に空行）
本日はお越しいただき
ありがとうございます。

本日のテーマを
ご説明します。

よろしくお願いします。
✅ 改行区切り（1行 = 1ブロック）
本日はお越しいただきありがとうございます。
本日のテーマをご説明します。
よろしくお願いします。
OUT
出力ファイルの保存先
📁 デスクトップ
　└── 📁 ハコ割り生成ツール_output/
　　　　├── output.srt ← 字幕ファイル（Premiere / DaVinci / YouTube 対応）
　　　　├── output.txt ← テキスト確認用
　　　　└── output.csv ← Excel で開ける表形式
⚠
困ったときは
症状 対処
ブラウザが開かない ブラウザのアドレスバーに http://127.0.0.1:8503 を直接入力してください
Docker エラーが出る Docker Desktop が起動しているか確認してください（🐳 アイコンが出ていること）
処理が遅い ローカル処理は CPU のみ使用します。60 分動画で約 15〜25 分が目安です
文字化けする テキストファイルを UTF-8 で保存し直してください
Mac で「開発元を確認できない」 アプリを右クリック →「開く」を選択してください（初回のみ）

詳しくは マニュアル.html をご覧ください。
