🚀 クイックセットアップ

はじめての方向け。インストールから最初の字幕生成まで 4 ステップで完了します。
✅ Docker 不要（Windows） 📄 SRT 字幕を自動生成 📝 原稿なし文字起こしも可 99 言語対応
1

インストール

インストーラーを実行するだけ。Docker・Python の事前準備は不要（Windows）

🪟 Windows
  • 配布された ハコ割り生成ツール_インストーラー.exe をダブルクリックするインストール先を聞かれたらそのまま「次へ」で OK
  • 「このアプリがデバイスに変更を加えることを許可しますか？」→ はい をクリックする
  • インストール完了後、デスクトップに「ハコ割り生成ツール（起動）」ショートカットが作成される
🍎 Mac（Apple Silicon）
  • 配布された ハコ割り生成ツール_setup.dmg をダブルクリックする
  • 開いたウィンドウで「ハコ割り生成ツール.app」を Applications フォルダにドラッグする
  • Applications から「ハコ割り生成ツール.app」を 右クリック → 「開く」 で起動する最初だけ「開発元を確認できない」と表示されます。「開く」をクリックすれば OK。2 回目以降は普通にダブルクリックで起動できます。
✅
Python・ffmpeg・Whisper AI モデルはすべて同梱されています Windows 版は追加インストール不要です。Mac 版は初回起動時のみインターネット接続が必要です（Whisper turbo モデルをダウンロードします。約 1.5 GB）。
2

ツールを起動する

ブラウザが自動的に開き、操作画面が表示されます

🪟 Windows
  • デスクトップの「ハコ割り生成ツール（起動）」アイコンをダブルクリックする黒いウィンドウが現れた後、ブラウザが自動で開きます（最大 30 秒ほど待ってください）
  • ブラウザが自動で開かない場合は、ブラウザのアドレスバーに http://127.0.0.1:8503 と入力して開く
🍎 Mac
  • Applications の「ハコ割り生成ツール.app」をダブルクリックする
  • ターミナルが起動し、しばらくするとブラウザが自動で立ち上がります
  • ブラウザが開かない場合は http://127.0.0.1:8503 をアドレスバーに入力する

3

ファイルをアップロードして実行する

使いたいモードを選んでください

🔖 モード A：ハコ割り生成（原稿あり）
  • 「🔖 ハコ割り生成」タブを選択する
  • 左のアップロード欄に音声・映像ファイルをドロップする（MP3 / WAV / MP4 など）
  • 右のアップロード欄にハコ割りテキスト（.txt）をドロップする
  • 「▶ SRT を生成」ボタンを押す
📝 モード B：文字起こし（原稿なし）
  • 「📝 文字起こし（原稿なし）」タブを選択する
  • アップロード欄に音声・映像ファイルをドロップする
  • 「▶ 文字起こしを実行」ボタンを押す

ハコ割りテキストの書き方（モード A のみ）

テキストファイル（.txt）に字幕の原稿を書きます。1 ブロック＝字幕 1 枚分です。

── 空行区切り（推奨）── 空行でブロックを区切ります 本日は皆様にお集まりいただき ありがとうございます。 本セミナーでは 3 つのテーマを取り上げます。 まず最初に ツールの基本的な使い方を ご説明します。
── 改行区切り── 1 行 = 1 ブロック 本日は皆様にお集まりいただきありがとうございます。 本セミナーでは 3 つのテーマを取り上げます。 まず最初にツールの基本的な使い方をご説明します。
⚠️
テキストファイルは UTF-8 で保存してください Windows のメモ帳：「ファイル」→「名前を付けて保存」→「エンコード」を UTF-8 に変更して保存。文字化けが起きる場合は必ずここを確認してください。
💡
まず書き起こしてから原稿を作る方法 「📝 文字起こし（原稿なし）」で全文を書き起こす → 出力された transcribe.txt をテキストエディタで編集してハコ割りを作る → 「🔖 ハコ割り生成」で SRT を生成する。原稿がまだない段階でも使えます。
4

完了後にファイルをダウンロードする

SRT・TXT・CSV の 3 ファイルが生成されます

処理が完了すると画面下にダウンロードボタンが 3 つ表示されます。

🔖 ハコ割り生成モードの出力ファイル
  • output.srt — Premiere Pro / DaVinci Resolve / YouTube に読み込む字幕ファイル
  • output.txt — 内容確認・テキストとして使えるファイル
  • output.csv — Excel で開けるタイムスタンプ一覧表
📝 文字起こしモードの出力ファイル
  • transcribe.srt — SRT 字幕ファイル
  • transcribe.txt — 議事録下書きや原稿作成に使えるテキスト
  • transcribe.csv — 行ごとに翻訳・編集しやすい CSV

処理時間の目安：

  • 5 分の音声 → 約 1〜2 分（ローカル処理 / turbo モデル）
  • 60 分の音声 → 約 15〜25 分
  • OpenAI API モード（サイドバーで切替）を使うと約 4〜8 倍速くなります（有料・要 API キー）
📁
出力ファイルの保存場所 デスクトップの ハコ割り生成ツール_output フォルダにも自動保存されます。ダウンロードボタンからも保存できます。
  • 音声ファイルを用意した（MP3 / WAV / MP4 など）
  • ハコ割りテキストを用意した（モード A のみ・UTF-8 で保存済み）
  • ブラウザでツールを開いた（http://127.0.0.1:8503）
  • ファイルをアップロードしてボタンを押した
  • 完了後にダウンロードボタンで SRT を保存した
✅
セットアップ完了！ タイミングの細かい調整、言語の切り替え、活用事例など詳しい情報は 詳細マニュアル をご覧ください。

よくあるつまずき

起動しない・エラーが出たときはここを確認してください

インストーラーを実行しても何も起きない・ブロックされる
ウイルス対策ソフトがブロックしている可能性があります。ウイルス対策ソフトの通知を確認し「許可する」または「除外」に設定してから再実行してください。
起動してもブラウザが開かない
  1. 黒いウィンドウが出たら 30 秒ほど待ってから、ブラウザで http://127.0.0.1:8503 を開く
  2. 黒いウィンドウが出ずに閉じた場合は再インストールを試みる
「▶ SRT を生成」ボタンが押せない（グレーになっている）
  • 「🔖 ハコ割り生成」タブ：音声ファイルとハコ割りテキストの両方をアップロードするとボタンが押せます
  • 「📝 文字起こし」タブ：音声ファイルだけアップロードするとボタンが押せます
「文字化けする」「ファイルが読み込まれない」
ハコ割りテキストを UTF-8 で保存し直してください。メモ帳の場合：「ファイル」→「名前を付けて保存」→ エンコードを UTF-8 に変更して保存。
Mac で「開発元を確認できない」と表示されて開けない
「ハコ割り生成ツール.app」を 右クリック → 「開く」 を選択し、ダイアログで「開く」をクリックしてください。2 回目以降は通常のダブルクリックで起動できます。
処理に時間がかかりすぎる
  • サイドバーの「Whisperモデル」を turbo（推奨）に設定されているか確認する
  • さらに高速化したい場合はサイドバーで OpenAI API モード（有料）に切り替える
  • 長い動画（60 分以上）は CPU 処理だと時間がかかります。処理中はそのまま待ってください
