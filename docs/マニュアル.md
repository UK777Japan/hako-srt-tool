## 1. ツールの概要

### ハコ割り生成ツールとは

動画や音声に字幕を付ける作業では、まず「ハコ割り」と呼ばれる字幕原稿を用意します。  
ハコ割りとは、セリフをどの単位で表示するかを決めたテキストのことです。

このツールは、**音声・映像ファイル** と **ハコ割りテキスト** をアップロードするだけで、  
Whisper AI が音声を認識し、各ハコのタイムスタンプを自動計算して **SRT 字幕ファイル** を生成します。

```
【従来の作業】
  ハコ割り作成 → 動画を見ながら1行ずつタイムスタンプを手入力 → 確認・修正
  （60分動画で 3〜5 時間かかることも）

【このツール使用後】
  ハコ割り作成 → ファイルをアップロードしてボタンを押す → 完了
  （60分動画で 15〜25 分）
```

### 対応ファイル形式

| 種類 | 対応形式 |
|------|---------|
| 音声 | MP3, WAV, M4A, AAC, FLAC, OGG |
| 映像 | MP4, MPG, MPEG, TS（音声トラックを自動抽出） |
| テキスト | TXT（UTF-8） |
| 最大ファイルサイズ | 10 GB |

### 出力ファイル

| ファイル | 説明 |
|---------|------|
| output.srt | SRT 字幕ファイル。Premiere Pro・DaVinci Resolve・YouTube などに対応 |
| output.txt | SRT と同じ内容のテキスト版（内容確認用） |
| output.csv | タイムスタンプ付き表形式（Excel で開ける） |

## 2. システム要件

| 項目 | 最小 | 推奨 |
|------|------|------|
| OS | Windows 10 / macOS 10.15 Catalina | Windows 11 / macOS 13 Ventura 以降 |
| RAM | 8 GB | 16 GB 以上 |
| ストレージ空き容量 | 5 GB | 10 GB 以上 |
| CPU | 4 コア | 8 コア以上 |
| インターネット | 初回のみ必要 | — |
| **Docker Desktop** | **必須（無料）** | 最新版 |

> **GPU について**  
> GPU がなくても CPU だけで動作します。ただし長い動画（30 分以上）はCPU のみだと  
> 処理に時間がかかります。速さを優先したい場合は OpenAI API モードをご利用ください。

## 3. インストール手順

### 3.1 Windows

#### ステップ 1: Docker Desktop のインストール

1. ブラウザで https://www.docker.com/products/docker-desktop/ を開く
2. **「Download for Windows」** をクリックしてダウンロード
3. ダウンロードした `Docker Desktop Installer.exe` をダブルクリック
4. インストール中に「Use WSL 2 instead of Hyper-V」の選択肢が出る場合は  
   そのまま（チェックあり）で **OK**
5. インストール完了後、PC を再起動するよう促された場合は再起動
6. スタートメニューから **Docker Desktop** を起動
7. タスクバー右下にクジラのアイコン 🐳 が表示されれば準備完了

> **確認方法**: コマンドプロンプトを開いて `docker --version` と入力し、

## 7. 入力ファイルの準備方法

### 音声・映像ファイル

**そのままアップロードできる形式:**

| 形式 | 説明 |
|------|------|
| MP3, WAV, FLAC | 音声ファイル。そのままアップロード |
| MP4 | 一般的な動画形式。音声トラックが自動的に抽出される |
| TS | 放送録画やキャプチャボードの出力形式。そのままアップロード |
| MPG / MPEG | 古い形式の動画。そのままアップロード |
| M4A, AAC | スマホや IC レコーダーの録音ファイル |

**品質のポイント:**
- 音声がクリアなほど認識精度が上がります
- 大きな BGM や効果音が重なっている場合は精度が下がることがあります
- 日本語の場合は言語設定を「Japanese / 日本語」にすると精度が上がります

---

### ハコ割りテキストの作り方

#### 文字コードについて

テキストファイルは **UTF-8** で保存してください。

**メモ帳（Windows）で保存する場合:**
1. ファイル → 「名前を付けて保存」
2. 「文字コード」を **「UTF-8」** に変更
3. 「保存」

**TextEdit（Mac）で保存する場合:**
1. フォーマット → 「標準テキストにする」
2. 「.txt で保存」を選択

#### 空行区切り形式（推奨）

字幕ブロックの間に空行を入れます。1 ブロック内は複数行でも OK です。

```
おはようございます。
本日は良いお天気ですね。

今日は３つのことを
お話しします。

最初のテーマは
「売上向上のための
戦略について」です。

まず、昨年の実績から
振り返ってみましょう。
```

#### 改行区切り形式

1 行 = 1 字幕ブロックとして扱います。

```
おはようございます。本日は良いお天気ですね。
今日は３つのことをお話しします。
最初のテーマは「売上向上のための戦略について」です。
まず、昨年の実績から振り返ってみましょう。
```

#### よくある失敗例と対策

| 失敗例 | 原因 | 対策 |
|--------|------|------|
| 処理後に字幕の数が少ない | 空行が入っておらず全体が 1 ブロックになっている | 区切りを正しく入れる |
| 文字化けする | Shift-JIS で保存されている | UTF-8 で保存し直す |
| タイミングが大きくずれる | 音声とテキストの内容が一致していない | テキストが音声の書き起こしか確認する |

## 8. 詳細設定リファレンス

### 音声認識エンジン

#### ローカル処理（推奨・デフォルト）

PC 上の Whisper AI を使って処理します。

- **無料**（追加費用なし）
- **インターネット不要**（2 回目以降）
- **ファイルサイズ制限なし**（上限 10 GB）
- 処理速度: CPU 環境で音声 1 分あたり約 1〜2 分

#### OpenAI API モード

OpenAI のサーバーで処理します。

- **有料**（$0.006 / 分）
- API キーが必要
- ファイルは **25 MB 以内**に圧縮（ツールが自動変換）
- 処理速度: 音声 1 分あたり約 5〜10 秒

**コスト例:**
| 動画の長さ | 費用の目安 |
|-----------|-----------|
| 5 分 | 約 $0.03（約 5 円） |
| 30 分 | 約 $0.18（約 28 円） |
| 60 分 | 約 $0.36（約 55 円） |

**API キーの入力方法:**
1. サイドバーで「OpenAI API（高速・要APIキー）」を選択
2. 表示されたテキストボックスに API キーを入力（`sk-...` から始まる文字列）
3. 入力したキーは自動保存され、次回以降は入力不要

---

### Whisper モデル

ローカル処理モード時に使用するモデルを選択できます。

| モデル | 精度 | 処理速度 | 用途 |
|--------|------|---------|------|
| **turbo（デフォルト）** | ★★★★★ | ★★★★★ | 通常はこれで十分 |
| large-v3 | ★★★★★ | ★★ | 精度最優先のとき |
| medium | ★★★★ | ★★★ | バランス重視 |
| small | ★★★ | ★★★★ | 試し処理に |
| base / tiny | ★★ | ★★★★★ | 超高速・精度は低い |

---

### 認識言語

音声の言語を選択します。正確な言語を指定すると精度が上がります。

| 設定 | 説明 |
|------|------|
| **Japanese / 日本語（デフォルト）** | 日本語に最適化 |
| 自動検出 | 自動で言語を判定（やや精度が下がる場合あり） |
| その他 99 言語 | 英語・中国語・韓国語など ABC 順で選択可能 |

---

### タイミング設定

#### グローバルオフセット（秒）

全タイムスタンプに一律で加算するオフセットです。  
生成した SRT を再生して「全体的に字幕が早い / 遅い」と感じた場合に調整します。

```
【例】字幕が音声より 0.3 秒遅れて表示される場合
  → グローバルオフセットを「-0.30」に設定して再生成

【例】字幕が音声より 0.5 秒早く表示される場合
  → グローバルオフセットを「+0.50」に設定して再生成
```

**キャッシュ機能との組み合わせ:**  
「書き起こしキャッシュを使用」にチェックが入っていると、  
再生成時に Whisper の音声認識はスキップされ（数秒で完了）、  
タイミング計算のみやり直します。とても便利な機能です。

#### 後端パディング（秒）

各字幕ブロックの終了時刻を何秒延ばすかを指定します。  
デフォルトの 0.20 秒は「少し余裕を持たせる」という設定です。

```
【例】セリフの終わりの余韻を長くしたい場合 → 0.50 に増やす
【例】テンポよく字幕を切り替えたい場合    → 0.10 に減らす
```

#### 最小表示時間（秒）

各字幕ブロックを最低何秒間表示するかを指定します。  
短いブロックが一瞬しか表示されないのを防ぎます。

#### 無音閾値（ms）

この設定時間（ミリ秒）より短い無音区間を無視して、  
前のブロックの表示を延長します。  
自然な「間」でブロックが切れすぎるのを防ぎます。

## 12. トラブルシューティング

### アプリが起動しない

**症状:** 起動ショートカットを実行してもブラウザが開かない。

**確認と対処:**

1. Docker Desktop が起動しているか確認する
   - Windows: タスクバー右下に 🐳 アイコンがあるか
   - Mac: メニューバー右上に 🐳 アイコンがあるか
   - ない場合は Docker Desktop を起動してから再度試す

2. コマンドウィンドウ（ターミナル）のエラーメッセージを確認する
   - 「Docker Desktop がインストールされていません」→ Docker Desktop をインストール
   - 「port is already allocated」→ 既に起動している。`http://localhost:8503` を開く

3. ブラウザを手動で開いて `http://localhost:8503` を入力する

---

### 初回起動でビルドが失敗する

**症状:** 「ビルドに失敗しました」と表示される。

**対処:**
- インターネットが接続されているか確認
- Docker Desktop を最新バージョンに更新
- ウイルス対策ソフトが Docker の通信をブロックしていないか確認
- 少し時間を置いてから再試行（Docker Hub のサーバー負荷の場合）

---

### 音声認識の精度が低い

**考えられる原因と対処:**

| 状況 | 対処 |
|------|------|
| 言語設定が違う | サイドバーで音声の言語を正しく選択 |
| BGM や雑音が多い | 事前に音声を編集してノイズを除去 |
| 音量が小さい | 音声を正規化（normalize）してからアップロード |
| 速い話し方や方言 | `large-v3` モデルまたは OpenAI API を試す |

---

### 字幕のタイミングがずれる

**全体的にずれる場合:**
→ グローバルオフセットで調整（[事例 5](#事例-5-タイミングを微調整する) 参照）

**特定のブロックだけずれる場合:**
→ そのブロック付近のハコ割りテキストと音声の内容が一致しているか確認

**字幕が早く消えてしまう場合:**
→ 「後端パディング」を 0.20 → 0.50 に増やす

---

### テキストが文字化けする

**症状:** ハコ割りテキストが正しく読み込まれず、文字化けしている。

**対処:**
- テキストファイルを UTF-8 で保存し直す（[7章 参照](#7-入力ファイルの準備方法)）

---

### ポート 8503 が使用中のエラー

**症状:** 「port is already allocated」のエラー。

**対処:**
1. 「停止」ショートカットを実行してから再起動
2. または、既に起動しているので `http://localhost:8503` に直接アクセス

## 13. よくある質問（FAQ）

**Q: 費用はかかりますか？**  
A: ローカル処理モードは完全無料です。  
OpenAI API モードを使う場合のみ OpenAI への支払いが発生します（$0.006/分）。

---

**Q: 処理したファイルはどこかに送られますか？**  
A: ローカル処理モードでは音声データは一切外部に送信されません。  
完全にプライベートな処理です。  
OpenAI API モードを使用した場合は、音声データが OpenAI に送信されます。

---

**Q: 処理時間の目安は？**

| 動画の長さ | ローカル（CPU・turbo モデル） | OpenAI API |
|-----------|---------------------------|------------|
| 5 分 | 約 1〜2 分 | 約 30 秒 |
| 30 分 | 約 5〜10 分 | 約 2〜3 分 |
| 60 分 | 約 15〜25 分 | 約 4〜6 分 |
| 120 分 | 約 30〜50 分 | 約 8〜12 分 |

---

**Q: 同じ音声ファイルで字幕テキストだけ変えて再生成できますか？**  
A: できます。「書き起こしキャッシュを使用」にチェックが入っていれば  
Whisper の音声認識はスキップされ、数秒で再生成できます。

---

**Q: ハコ割りテキストがない場合は使えませんか？**  
A: ハコ割りテキストは必須です。  
テキストがない場合は、まずハコ割りを作成してからご利用ください。

---

**Q: 複数ファイルを一度に処理できますか？**  
A: 現在は 1 ファイルずつの処理です。  
複数ある場合は順番に処理してください。

---

**Q: アンインストールはどうすればいいですか？**

**Windows:**  
スタートメニュー →「ハコ割り生成ツール」→「アンインストール」

または、コントロールパネル → プログラムと機能 → 「ハコ割り生成ツール」→「アンインストール」

**Mac:**  
Finder → Applications → 「ハコ割り生成ツール.app」をゴミ箱へ

Docker イメージも削除する場合（ストレージ節約）：
```
アプリを停止してから、ターミナルで以下を実行:
docker rmi hako-srt-app
```

---

**Q: 出力フォルダの場所を変えられますか？**  
A: 現在の設定ではデスクトップの `ハコ割り生成ツール_output` フォルダに固定されています。  
OneDrive や Dropbox と同期したい場合は、そのフォルダをクラウドの同期対象に  
設定するのが最も簡単な方法です。
