📄 ハコ割り生成ツール　マニュアル

音声・映像ファイルとハコ割りテキストから SRT 字幕ファイルを自動生成するツールです。

99言語対応 最大10GB SRT / CSV 出力 ローカル処理（無料） OpenAI API モード
1

ツールの概要

ハコ割り生成ツールとは

「ハコ割り」とは、動画字幕の原稿として字幕をどの単位で表示するかを決めたテキストのことです。このツールは 音声・映像ファイル と ハコ割りテキスト をアップロードするだけで、Whisper AI が音声を認識し、各ハコのタイムスタンプを自動計算して SRT 字幕ファイル を生成します。

【従来の作業】 ハコ割り作成 → 動画を見ながら1行ずつタイムスタンプを手入力 → 確認・修正 （60分動画で 3〜5 時間かかることも） 【このツール使用後】 ハコ割り作成 → ファイルをアップロードしてボタンを押す → 完了 （60分動画で 15〜25 分）

対応ファイル形式

種類対応形式備考
音声MP3, WAV, M4A, AAC, FLAC, OGGそのままアップロード
映像MP4, MPG, MPEG, TS音声トラックを自動抽出
テキストTXT（UTF-8）ハコ割りテキスト
最大サイズ10 GB

出力ファイル

📝 output.srt
SRT 字幕ファイル
  • Premiere Pro
  • DaVinci Resolve
  • YouTube / Vimeo
  • その他 SRT 対応ツール
📄 output.txt
タイムスタンプ付きテキスト
  • 内容確認用
  • SRT と同じフォーマット
📊 output.csv
表形式データ
  • Excel で開ける
  • No / Start / End / Duration / Text
  • タイミング確認に便利

インストール手順

Windows へのインストール（Docker 不要）

事前準備は不要です Python・ffmpeg・Whisper AI モデル（turbo）はすべてインストーラーに同梱されています。Docker Desktop は必要ありません。

1
ハコ割り生成ツール_setup.exe をダブルクリック「このアプリがデバイスに変更を加えることを許可しますか？」→「はい」
2
セットアップウィザードが起動します「次へ」を数回クリックしてインストール完了（インストール先の変更は不要です）
3
「デスクトップにショートカットを作成」にチェックが入っていることを確認 →「インストール」
4
完了画面で「ハコ割り生成ツールを今すぐ起動する」にチェックを入れたまま「完了」ブラウザが自動で開いたら準備完了です。
インストール後に作成されるもの デスクトップ:「ハコ割り生成ツール 起動」「ハコ割り生成ツール 停止」のショートカット
スタートメニュー:「ハコ割り生成ツール」グループ（起動・停止・マニュアル・アンインストール）

ステップ 2: Docker Desktop と ハコ割り生成ツールをインストール

基本的な使い方

1
音声・映像ファイルをアップロード 「音声・映像ファイル」エリアをクリックしてファイルを選択、またはドラッグ&ドロップします。
対応形式: MP3 / WAV / MP4 / MPG / MPEG / TS / M4A / AAC / FLAC / OGG
2
ハコ割りテキストをアップロード UTF-8 で保存されたテキストファイル（.txt）をアップロードします。
3
区切り方式を選択 テキストの形式に合わせて「空行区切り」または「改行区切り」を選びます（詳細は7章）。
4
「▶ SRT を生成」をクリック 処理が始まり、ステップごとの進捗がリアルタイムで表示されます。
5
ダウンロード 完了後に output.srt output.txt output.csv の 3 ボタンが表示されます。クリックするとダウンロードできます。
同じファイルはデスクトップの「ハコ割り生成ツール_output」フォルダにも保存されます。

入力ファイルの準備方法

ハコ割りテキストの文字コード

テキストファイルは必ず UTF-8 で保存してください。

ソフト保存方法
メモ帳（Windows）ファイル → 名前を付けて保存 →「文字コード」を UTF-8 に変更して保存
TextEdit（Mac）フォーマット →「標準テキストにする」→「.txt で保存」を選択
VS Code / テキストエディタ右下の文字コード表示をクリック → UTF-8 を選択して保存

空行区切り形式（推奨）

字幕ブロックの間に空行（改行のみの行）を入れます。1 ブロック内は複数行でも問題ありません。

おはようございます。 本日は良いお天気ですね。 ← 空行 今日は３つのことを お話しします。 ← 空行 最初のテーマは 「売上向上のための 戦略について」です。

改行区切り形式

1 行 = 1 字幕ブロックとして扱われます。インタビューや会話形式に適しています。

おはようございます。本日は良いお天気ですね。 今日は３つのことをお話しします。 最初のテーマは「売上向上のための戦略について」です。

よくある失敗と対策

症状原因対策
字幕の数が少ない（1〜2個しかない）区切りが入っておらず全体が1ブロックになっている空行区切り or 改行区切りを正しく選択
文字化けするShift-JIS で保存されているUTF-8 で保存し直す
タイミングが大きくずれる音声の内容とテキストが一致していないテキストが音声の書き起こしか確認する

タイミング設定

グローバルオフセット（秒）
デフォルト: 0.00
全タイムスタンプに一律で加算するオフセット。字幕が全体的に早い・遅い場合に調整します。
例: 字幕が 0.3 秒遅れる → -0.30 に設定
後端パディング（秒）
デフォルト: 0.20
各ブロックの終了時刻を延ばす秒数。余韻を持たせたい場合は増やします（例: 0.50）。
最小表示時間（秒）
デフォルト: 0.80
各ブロックを最低でも何秒間表示するかを指定します。短いブロックが一瞬で消えるのを防ぎます。
無音閾値（ms）
デフォルト: 400
この時間より短い無音区間は前のブロックを延ばして埋めます。自然な間での区切れを防ぎます。
キャッシュを活用したタイミング再調整 「書き起こしキャッシュを使用」にチェックが入った状態でオフセットを変更して再生成すると、Whisper の音声認識がスキップされ数秒で完了します。

ブラウザのタブを閉じただけではアプリがバックグラウンドで動き続けます。不要な場合は停止してください。

OS停止方法

トラブルシューティング

アプリが起動しない / ブラウザが開かない
  1. タスクバーに「ハコ割り生成ツール」のウィンドウが最小化されていないか確認する（クリックして開くとエラーが確認できます）
  2. ブラウザで http://127.0.0.1:8503 を手動で開いてみる
  3. ウイルス対策ソフトが Python の動作をブロックしていないか確認する
  4. 再インストールを試みる
「Python が見つかりません」と表示される
  • インストールが正常に完了しているか確認する
  • 再インストールを試みる（古いバージョンをアンインストール後に再インストール）
  • インストール先フォルダ（%LOCALAPPDATA%\HakoSRTTool\python\）が存在するか確認する
音声認識の精度が低い
状況対処
言語設定が違うサイドバーで音声の言語を正しく選択する
BGM や雑音が多い事前に音声を編集してノイズを除去する
速い話し方や方言large-v3 モデルまたは OpenAI API を試す
小さいモデルを使用turbo または large-v3 に変更する
字幕のタイミングがずれる
  • 全体的にずれる: グローバルオフセットで調整（事例 5 参照）
  • 特定の箇所だけずれる: ハコ割りテキストと音声の内容が一致しているか確認する
  • 字幕が早く消える:「後端パディング」を 0.20 → 0.50 に増やす
Mac で「開発元を確認できない」と表示される
「ハコ割り生成ツール.app」を右クリック →「開く」を選択してください。表示されるダイアログで「開く」をクリックすれば起動できます（3章参照）。
テキストが文字化けする
ハコ割りテキストを UTF-8 で保存し直してください（7章参照）。

よくある質問（FAQ）

Q費用はかかりますか？
ローカル処理モードは完全無料です。OpenAI API モードを使う場合のみ OpenAI への支払いが発生します（$0.006/分）。
Q処理したファイルはどこかに送られますか？
ローカル処理モードでは音声データは一切外部に送信されません。完全にプライベートな処理です。OpenAI API モードを使用した場合は、音声データが OpenAI に送信されます。
Q処理時間の目安は？
動画の長さローカル（CPU・turbo）OpenAI API
5 分約 1〜2 分約 30 秒
30 分約 5〜10 分約 2〜3 分

Q同じ音声でテキストだけ変えて再生成できますか？
できます。「書き起こしキャッシュを使用」にチェックが入っていれば Whisper の音声認識はスキップされ、数秒で再生成できます。
Qインターネットなしで使えますか？
初回起動のみインターネットが必要です（Whisper モデルのダウンロード）。2回目以降は完全にオフラインで動作します。
Qアンインストール方法は？
Windows: スタートメニュー →「ハコ割り生成ツール」→「アンインストール」
Mac: Finder → Applications →「ハコ割り生成ツール.app」をゴミ箱へ

Docker イメージも削除する場合（ストレージ節約）:
アプリ停止後、ターミナルで docker rmi hako-srt-app を実行
